## Run locally

- Clone the project

```bash
  git clone https://github.com/MrAzuka/LibMag.git
```

- Go to the project directory

```bash
  cd libmag
```

- Install dependencies

```bash
  npm install
```
- Copy .env.example to .env

- Run the application

```bash
  npm run dev
```
